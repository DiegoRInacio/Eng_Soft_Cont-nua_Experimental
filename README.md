# Simulado Integrado de Engenharia de Software

Projeto estatico em HTML para disponibilizar, via GitHub Pages, um simulado integrado com questoes objetivas e discursivas.

O simulado cobre os temas:

- Desenvolvimento orientado a reuso de software
- Engenharia de requisitos
- Gestao de projetos de TI
- Linguagens formais e automatos
- Qualidade de software

O arquivo principal do site e o `index.html`. O repositorio foi configurado para versionar apenas os arquivos necessarios para publicacao no GitHub Pages.
